# 💣 Bomb Defusal Workshop

เกมกู้ระเบิดกระดาษ สำหรับสอนแนวคิดการเขียนโปรแกรมพื้นฐาน
ออกแบบมาสำหรับนักเรียนมัธยมที่ไม่เคยเขียนโค้ดมาก่อน
ใช้ในงาน Intania Openhouse 2026 โดย Workshop ภาควิชาวิศวกรรมคอมพิวเตอร์ จุฬาลงกรณ์มหาวิทยาลัย

---

## 🎮 เกมนี้สอนอะไร?

| ด่าน | ชื่อ | แนวคิด |
|------|------|--------|
| 1 | หุ่นยนต์บุกโรงงาน | Sequence, Function |
| 2 | กู้ระเบิดสายไฟ | If / Else |
| 3 | ระเบิดทายรหัสผ่าน | Binary Search |

---

## 👥 รูปแบบการเล่น

แบ่งผู้เล่นเป็นทีม 5 คน แบ่งบทบาทเป็น:

- **Programmer (3 คน)** — เห็นข้อมูลและโจทย์ทั้งหมด
  ต้องเขียน algorithm ส่งให้ Player
- **Player (2 คน)** — ไม่เห็นข้อมูลใดๆ ล่วงหน้า
  รับ algorithm จากทีมแล้วลงมือแก้ระเบิดตามนั้น

*ตอนรันจริง Player จะรอโปรแกรมด่าน 2 นานมาก ถ้ามีเวลาจำกัด ให้ Player แก้ด่าน 3 ไปก่อนได้

## 🚀 วิธีใช้งาน

### สิ่งที่ต้องเตรียม

- คอมพิวเตอร์ 1 เครื่องต่อกลุ่ม (สำหรับเกม)
- ปริ้น Instruction ทั้ง 2 ฝั่งต่อด่าน
- (ด่าน 1–2) card คำสั่งกระดาษ + กาวสองหน้า + futureboard สำหรับวาง card

### วิธีรันเกม

ไม่ต้องติดตั้ง — เปิด HTML ใน browser ได้เลย
````bash
# clone repo
git clone https://github.com/daykungza/openhouse-comprog-game.git

# เปิดเกมด่านที่ต้องการ
open game/stage2-wires.html
````
