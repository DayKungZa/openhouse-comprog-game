# 💣 Com Prog Workshop

สายลับ intania กับภารกิจกู้ระเบิดที่โรงงาน IronHaven Factory
ใช้ในงาน Intania Openhouse 2026 โดย Workshop ภาควิชาวิศวกรรมคอมพิวเตอร์ จุฬาลงกรณ์มหาวิทยาลัย

เกมนี้สำหรับสอนนักเรียนมัธยมที่อาจจะมีหรือไม่มีพื้นฐานเขียนโค้ดมาก่อน

---

## 🎮 เกมนี้สอนอะไร?

| ด่าน | ชื่อ | แนวคิด |
|------|------|--------|
| 1 | หุ่นยนต์บุกโรงงาน | Sequence, Function |
| 2 | กู้ระเบิดสายไฟ | If / Else |
| 3 | ระเบิดทายรหัสผ่าน | Binary Search |

---

## 👥 รูปแบบการเล่น

แบ่งผู้เล่นเป็นทีม 5 คน แบ่งบทบาทเป็น:

- **Programmer (3 คน)** — เห็นข้อมูลและโจทย์ทั้งหมด
  ต้องเขียน algorithm ส่งให้ Player
- **Player (2 คน)** — ไม่เห็นข้อมูลใดๆ ล่วงหน้า
  รับ algorithm จากทีมแล้วลงมือแก้ระเบิดตามนั้น

| ช่วง | เวลา |
|------|------|
| ด่าน 1 | 10 นาที |
| ด่าน 2+3 | 15 นาที |

*ด่าน 3 ใช้เวลาไม่เกิน 5 นาที

## 📂 ไฟล์ที่มีในนี้

| ไฟล์ | คือ |
|------|------|
| `game1_robot.html` | เกมด่าน 1 หุ่นยนต์บุกโรงงาน |
| `game2_wires.html` | เกมด่าน 2 กู้ระเบิดสายไฟ |
| `game3_binary.html` | เกมด่าน 3 ระเบิดพิสดาร |
| `instruction.pdf` | Instruction รวม |
| `program_card.pdf` | ไฟล์สำหรับพิมพ์การ์ดโปรแกรมกระดาษ |

โดยในไฟล์ instruction จะแบ่งเป็น
- ด่าน 1 for programmer (หน้า 1-2)
- ด่าน 1 for player (หน้า 3)
- ด่าน 2 for programmer (หน้า 4-5)
- ด่าน 2 for player (หน้า 6)
- ด่าน 3 for nong (หน้า 7)

## 🚀 วิธีใช้งานเกม

### สิ่งที่ต้องเตรียม

- คอมพิวเตอร์ 1 เครื่องต่อกลุ่ม (สำหรับเกม)
- ปริ้น Instruction ทั้ง 2 ฝั่งต่อด่าน
- (ด่าน 1–2) card คำสั่งกระดาษ + กาวสองหน้า + futureboard สำหรับวาง card

### วิธีรันเกม

ไม่ต้องติดตั้ง — เปิด HTML ใน browser ได้เลย
````bash
# clone repo
git clone https://github.com/daykungza/openhouse-comprog-game.git

# เปิดเกมด่านที่ต้องการ
open game/stage2-wires.html
````
